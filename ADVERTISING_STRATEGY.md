## UTM Parameter Convention

All paid traffic must use consistent UTM parameters for attribution:

| Parameter | Format | Example |
|---|---|---|
| utm_source | Platform name (lowercase) | `google`, `tiktok`, `snapchat` |
| utm_medium | Ad type | `cpc`, `paid_social`, `display` |
| utm_campaign | Campaign slug | `ai-assistant-au`, `cyber-tools-pro` |
| utm_content | Ad variant | `ad-group-1`, `video-1`, `snap-1` |
| utm_term | Keyword (Google only) | `ai+assistant+australia` |

### Example UTM URLs
```
https://archibaldtitan.com/?utm_source=google&utm_medium=cpc&utm_campaign=ai-assistant-au&utm_content=ad-group-1
https://archibaldtitan.com/?utm_source=tiktok&utm_medium=paid_social&utm_campaign=ai-works&utm_content=video-1
https://archibaldtitan.com/?utm_source=snapchat&utm_medium=paid_social&utm_campaign=ai-gen-z&utm_content=snap-1
```

## Performance Tracking & KPIs

### Weekly Review Metrics
- Total spend vs budget
- CTR by platform and ad group
- CPC by platform
- Conversion rate by source
- Cost per Conversion (CPA)
- ROAS (Return on Ad Spend)

### Monthly Review
- Platform budget reallocation based on ROAS
- Creative refresh (new Video/Image variants)
- Keyword expansion/pruning (Google)
- Audience refinement (TikTok, Snapchat)

### Conversion Goals
1. Free signup (primary)
2. Pro plan upgrade (secondary)
3. Titan plan upgrade (tertiary)

## Creative Guidelines
